Background

The app visualizes Chicagos 2017 crime data. The data set used in this assignment is availble at the Chicago Data Portal and was extracted from the Chicago Police Departments CLEAR (Citizen Law Enforcement Analysis and Reporting) system. This project is a continuation of Assignment #2 for Courseras Developing Data Products course.

The shiny app for this project can be found here.

The code for the app is located in this github repo.

Shiny Application

The user can visualize the data after selecting a crime type (up 5 for the line graph and scatter plot).
